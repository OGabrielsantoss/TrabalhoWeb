Perguntas Frequentes

O que é HTML?

HTML é a linguagem usada para estruturar páginas na web.

Para que serve o CSS?

O CSS é usado para estilizar páginas, como cores, layouts e fontes.

O que é CSS Grid?

É um sistema de layout que permite criar grades responsivas.

Preciso saber JavaScript?

Não é obrigatório no início, mas ajuda a deixar o site interativo.
