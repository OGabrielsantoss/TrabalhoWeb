## Objetivo

Este repositório reúne a resolução completa dos exercícios propostos na Lista 01 do 1º bimestre, com foco no desenvolvimento e aprimoramento das habilidades em HTML5 e CSS3.

Ao longo dos exercícios, são explorados conceitos fundamentais da construção de páginas web, incluindo estruturação semântica, organização de conteúdo e estilização visual, utilizando diferentes tags HTML e propriedades CSS específicas exigidas em cada atividade.

Cada exercício foi desenvolvido com o objetivo de consolidar o aprendizado prático, aplicando boas práticas de codificação, organização de arquivos e clareza na construção das interfaces, além de preparar o aluno para situações reais no desenvolvimento front-end.
